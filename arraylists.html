Arraylists

Concept Theory

Arraylists are very similar to arrays, except are resizable and therefore, a little easier to use in practice. However, Java can only story objects (including Strings) in arraylists, so primitive data types such as ints and floats can not be stored. Here are some of the functions of an arraylist:

Declaring an arraylist without values

ArrayList names = new ArrayList();

This creates a new string arraylist called names.

Declaring an arrraylist with values

.List names = Arrays.asList( "Julie", "George"); method adds whatever is in the brackets to the ned of the array.

This creates a new string arrayList with names Julie and George.

Adding values to the list

.add(); method adds whatever is in the brackets to the ned of the array.

names.add("Henry"); --> will add Henry to the end of the array

Accessing a value

.get(n); where n is the index of the value.

System.out.println(names.get(0)); --> will print Julie

Removing values

.remove(n); where n is the index of the value being removed

names.remove(0); --> will remove Julie

.clear --> will remove all contents from the array

Length of the list

.size(); will give the lenght of the arraylist

System.out.println(names.size()); --> will print 3

Looping through an arraylist using enhanced for loops

for (int name: names) {

System.out.println(name); }

This for each loop will print out the names in the name array one by one.

Common errors when using arraylists
  • ArrayIndexOutOfBounds - If a for loop increments past the final index of an arraylist (length-1), it will crash.
  • Off-By-One - We naturally start counting at 1, but the first index of an arraylist is 0.

evaluateHand(ArrayList hand) --> contains several if statements that calculate the total of a certain person, depending on their cards.

displayHand(ArrayList points, ArrayList pHand, int pNum) --> displays a players cards in rows, with their total points

declareWinner --> (shown below) uses enhanced for loops and a arraylist argument, to go through all of the players' points and determine the top player.
