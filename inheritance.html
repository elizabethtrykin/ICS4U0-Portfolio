Inheritance

Concept Theory

Inheritance allows one class can inheret (very similar to real world inheritance) the attributes and behaviours of another class.

Polymorphism

Polymorphism explains that there is a differnce between compile time and run time. During compile time, when an object is created, the program will look at the type of the object on the left. So in the case below, it will consider he as person. However, it also sees that I'm trying to call the method cackle();. So, at compile time, Java looks for the method cackle(); in Person, but does not find it and throws a compile time error

Now if the code passes through compile time, the program will then run, and only now check if the subclass has the method cackle();. So, if you create a superclass type, but declare as a subclass and then running methods, make sure that the superclass also contains the method.
