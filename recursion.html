Recursion

Concept Theory

Recursion is a way to make methods call themselves, for an easier way to solve problems.

Every recursive method has to have a base and a recursive case. The base case is the case when the method will stop repeating itself, which makes sure that the method doesn't run infinately. The recursive cases are the cases when the method calls itself again.

Common mistakes are usually logic mistake. When coding a recursive method, you have to make sure that it is actually performing what you want it to be performing.
