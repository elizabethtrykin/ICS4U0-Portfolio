Arrays

Concept Theory

Arrays are used for storing a certain number of elements of the same data type. They are most useful for storing variables that corrolate with one another, so that they can be easily accessed. For example, I could store student names in an array.

How to declare an array
  • Declaring an array - String [] arr = new String[n]; (where n is the size of the array)
  • Initializing an array - String [] arr = new String{"Grace", "George", "Athena"};
  • Accessing an index of an array - System.out.println(arr[0]); (would print Grace);
  • Printing out an entire array:

for (int i = 0; i < 3; i++) {

System.out.println(arr[i]); }

Common errors when using arrays
  • ArrayIndexOutOfBounds - When for loop, if the loop increments beyond the last index of the array (length - 1), the program will crash.
  • Off-By-One - We naturally start counting at 1, by array indexes start at 0, which means that in a loop, the index would start at 0.
