Sorting

Concept Theory

Sorting is used to sort arrays of values from least to greatest, or vice versa or in alphabetical order. There are many different sorting alogirthms, and I'll outline some of them here.

Bubble sort

Works by swapping consequent values if they're not in the corect order. The loop wil start at the very beginning of the array, and go through every element, swapping them into the correct order.

Insertion sort

Works by splitting the array into sorted (starting the first element on the left) and unsorted (elements on the right). The loop starts with the leftmost element, which should either be the greatest or the smallest and compares every element on the right to that element. It will do so until there needs to be a swap.

Selection sort

This sorting method will look for the minimum/maximum value in the array, and add it to the left. It will do so until the entire array is sorted from least to greatest or vice versa.

Quicksort

Works by picking a random element in the array, and adds all the smaller values to the left, then all the larger values to the right. It then, does so with every single value until the array is sorted.

Quicksort

Works by picking a random element in the array, and adds all the smaller values to the left, then all the larger values to the right. It then, does so with every single value until the array is sorted.

Common errors when sorting through arrays:
  • ArrayIndexOutOfBounds - If a for loop increments past the final index of an array (length-1), it will crash.
  • Off-By-One - We naturally start counting at 1, but the first index of an array is 0.
